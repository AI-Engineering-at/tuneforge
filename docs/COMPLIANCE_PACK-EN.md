# TuneForge Compliance Pack
- Language: EN
- Audience: Public
- Last Sync: 2026-03-19
- Pair: COMPLIANCE_PACK-DE.md

TuneForge provides audit-ready engineering documentation for Austria, DSGVO, and the EU AI Act. It is **not legal advice** and it is not a compliance certification.

## Purpose

The public compliance pack helps teams prepare:

- model release evidence
- dataset provenance records
- benchmark traceability
- intended-use and limitation statements
- DPIA and VVT input packages
- release approval and rollback evidence

## Austria, DSGVO, and EU AI Act Position

- Austria: support for local governance review, release traceability, and source attribution
- DSGVO: support for provenance review, retention planning, and documentation of human review boundaries
- EU AI Act: support for intended-use framing, risk review inputs, technical documentation, and evidence retention

## What This Pack Does Not Do

- it does not approve a deployment
- it does not replace counsel, DPO review, or Austrian legal review
- it does not make a base model or dataset lawful by default
- it does not convert a preview model into a verified public claim without evidence

## Minimum Public Release Evidence

- model card
- data provenance record
- benchmark evidence record
- release approval decision
- environment manifest
- tester attestation
- validation result
- license manifest
